#  주소록 관리 프로그램 업그레이드 버전 (V2)

기존의 단순 리스트 구조를 탈피하여, **객체 지향 프로그래밍(OOP)**의 핵심 원칙인 상속과 다형성을 적용하고 **메모리 효율**을 극대화한 업그레이드 버전입니다.

---

##  주요 업그레이드 포인트

### 1. 객체 지향 설계 (상속 구조 도입)
- **기본 클래스(Addr)**: 이름, 전화번호, 이메일, 주소, 생일, 직급 등 공통 정보를 관리하는 부모 클래스입니다.
- **회사 연락처(CompanyAddr)**: Addr를 상속받아 회사명, 부서명을 추가로 관리합니다.
- **거래처 연락처(CustomerAddr)**: Addr를 상속받아 거래처명, 품목명을 추가로 관리합니다.
- **다형성(Polymorphism)**: super().print_info() 호출을 통해 코드 중복을 최소화하고 유지보수성을 높였습니다.

### 2. 메모리 최적화 및 데이터 관리
- **enumerate() 활용**: 검색, 삭제, 수정 시 인덱스를 정확히 추적하여 리스트 처리 성능을 최적화했습니다.
- **메모리 자원 해제**: 삭제 시 del 키워드를 사용하여 메모리에서 객체를 명시적으로 제거, 리소스를 효율적으로 관리합니다.

### 3. 프로그램 안정성 (Exception Handling)
- **예외 처리**: 메뉴 입력 시 발생할 수 있는 오류를 try-except로 방어하여 잘못된 입력에도 프로그램이 종료되지 않도록 설계했습니다.
- **코드 품질**: 모든 클래스와 메서드에 Docstring을 추가하여 가독성과 협업 효율을 높였습니다.

---

##  실행 방법
1. 파이썬이 설치된 환경에서 SmartPhoneMain.py를 실행합니다.
   - 명령어: python SmartPhoneMain.py
2. 메뉴(1~7)를 선택하여 주소록 기능을 이용합니다.
   - 1: 회사 연락처 등록
   - 2: 거래처 연락처 등록
   - 3: 전체 출력
   - 4: 검색
   - 5: 삭제
   - 6: 수정
   - 7: 프로그램 종료

##  파일 구성
- SmartPhoneMain.py: 모든 클래스와 로직이 포함된 통합 실행 파일
- README.md: 프로그램 상세 설명서
